## Quiz Card Function

기능 : 
1. pm 13:00 에 자체 제작한 질문지 질문 5가지 중 랜덤으로 사용자에게 질문 카드가 전송 
2. 질문지는 연인에 대한 정보를 맞추는 기능 (예시 : 요즘 최애 야식 메뉴는? 떡볶이)
3. 연인이 카드를 받고 정답/오답을 버튼으로 평가

1. Data flow

- 13시에 질문지 질문 프론트로 전송
- 수신인 답변 작성
- 답변 카드가 각자의 연인에게 전송
- 연인이 정답/오답 선택
- 정답/오답 시 캐릭터 애정도에 대한 점수 +/- 부여 (백엔드에서 점수로 연산)
- 백엔드에서 애정도 계산
- 계산 된 애정도를 프론트에 전송

## Praise Card Function

기능 및 Flow : 
1. 칭찬 카드의 문장을 작성 (프론트 -> 백엔드)
2. 백엔드 -> 연인에게 칭찬 카드 전송
3. 연인이 감동/그냥 평가를 백엔드 전송 (프론트 -> 백엔드)
4. 백엔드 : 감동 평가를 점수로 연산 후 애정도 증가 산출
5. 프론트로 애정도 증가 값을 전송
